# 🍕 Projeto Fullstack — Bella Pizza

Este repositório contém o desenvolvimento de uma aplicação fullstack construída durante o curso [“Projeto Fullstack do Zero”](https://www.udemy.com/course/projeto-fullstack-do-zero-react-node-postgresql/). O objetivo é aprender na prática como funciona o ciclo completo de uma aplicação desde a interface até o banco de dados, passando pela construção de APIs e integração entre camadas.
